# Marketing campign Data Analysis Project - Python

## 📊 Project Overview

This project focuses on performing exploratory data analysis (EDA) using Python to understand patterns, relationships, and trends within the dataset. The objective is to clean the data, explore important variables, and generate meaningful insights through statistical analysis and visualizations.

---

## 🎯 Problem Statement

Organizations collect large volumes of data, but extracting useful insights from this data requires proper analysis.
The goal of this project is to analyze the dataset, identify patterns, and visualize important relationships that can support data-driven decision-making.

---

## 📂 Dataset Description

The dataset used in this project contains structured data that includes multiple variables used for analysis.
The dataset allows exploration of relationships between different features and helps identify key patterns in the data.

Key attributes in the dataset include:

* Numerical variables
* Categorical variables
* Target or outcome variables used for analysis

---

## 🔧 Data Cleaning

Before analysis, the dataset was prepared by performing the following steps:

* Handling missing values
* Checking for duplicate records
* Correcting data types
* Preparing data for analysis

These steps ensure the dataset is reliable for accurate analysis.

---

## 📊 Exploratory Data Analysis (EDA)

Exploratory Data Analysis was performed to understand the distribution of variables and relationships between features.

The following types of analysis were performed:

* Distribution analysis of numerical variables
* Correlation analysis between variables
* Comparison between categorical variables
* Identification of trends and patterns

Several visualizations were created to better understand the dataset.

---

## 📈 Visualizations

The project includes multiple charts to illustrate key patterns in the dataset, including:

* Distribution plots
* Bar charts
* Correlation heatmap
* Feature comparison charts

These visualizations help highlight relationships and trends within the data.

---

## 🔍 Key Insights

Important insights discovered during the analysis include:

* Certain variables show strong relationships with the target variable.
* Some categories contribute significantly more to the overall distribution.
* Patterns in the data reveal trends that may influence business or analytical decisions.
